# selenium-project-with-java
This repository contains an example of how to build a selenium project with java on the website [Gadget & Gear](https://gadgetandgear.com/).

## Technologies:
- Selenium WebDriver
- Java
- Maven
---

### Dependencies
* selenium-java
* webdrivermanager 
* testng
* allure-testng
---
